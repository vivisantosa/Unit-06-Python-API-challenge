Python-API-challenge

The Python-API-challenge about generating information through Python script from external sources using API and JSON queries with Python.
This challenge is a two-parts project; WeatherPy and the VacationPy.



WeatherPy

The WeatherPy project is to analyze the relationship between the location of a city (defined by their longitude and latitude) and its weather characteristics. The weather patterns that we analyze are temperature, humidity, cloudiness, and wind speed.
The hypothesis is that cities with similar distance from the equator, have similar weather patterns. We create sample lists of cities that represent an area (eg Southern Hemisphere, The Tropics), queries the weather data for those cities, create visualization out of the weather data, perform regression analysis, and draw conclusion.
